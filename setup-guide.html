4-Week Launch Roadmap

Week 1
Foundation
Days 1–7
  • 1
    Register Domain $10.98
    • Go to namecheap.com → click Domain Search → type TechPackCage.com → Add to Cart → Checkout
    • Enable WhoisGuard Privacy (FREE with Namecheap) during checkout
  • 2
    Sign Up for Hosting $3.99/mo
    • Go to hostinger.com → click Hosting → select Business Shared Hosting
    • During signup, choose "I already have a domain" → enter TechPackCage.com
    • Complete payment → access hPanel dashboard
  • 3
    Point Domain to Hostinger
    • In Namecheap Dashboard → Domain List → TechPackCage.com → Manage → Nameservers → Custom DNS
    • Enter Hostinger nameservers:
    ns1.hostinger.com
    ns2.hostinger.com
    • Wait 24–48 hours for DNS propagation. Verify at: whatsmydns.net
  • 4
    Install WordPress
    • Login to Hostinger hPanel → Website → Auto Installer → WordPress
    • Fill in: Site Name TechPackCage, Admin Username, Strong Password, Email
    • Click Install → wait 2–3 minutes
  • 5
    Install WooCommerce
    • Login to WordPress Admin at yourdomain.com/wp-admin
    • Go to: Plugins → Add New → search WooCommerce → Install Now → Activate
    • Complete WooCommerce Setup Wizard: store location, currency (CAD or USD), payment methods
  • 6
    Install Astra Theme
    • Go to: Appearance → Themes → Add New → search Astra → Install → Activate
    • Go to: Appearance → Astra Options → import "Shop" starter template
  • 7
    Set Up Cloudflare CDN
    • Go to cloudflare.com → Create account → Add Site → enter TechPackCage.com
    • Cloudflare scans DNS → click Continue → select Free plan
    • Replace Namecheap nameservers with Cloudflare nameservers (format: arlo.ns.cloudflare.com)
    • In Cloudflare: SSL/TLS → set to "Full (Strict)"
  • 8
    Enable SSL
    • In Hostinger hPanel → SSL → Let's Encrypt → click Enable next to TechPackCage.com
    • Wait 10 minutes → verify https:// works in browser
    • In WordPress: Settings → General → change both URLs to https://
Week 2
Products
Days 8–14
  • 1
    Purchase AliDropship Plugin $89 one-time
    • Go to alidropship.com → click Get AliDropship → $89 one-time (lifetime license)
    • Download the plugin .zip file from your account dashboard
  • 2
    Install AliDropship
    • WordPress Admin → Plugins → Add New → Upload Plugin → choose .zip → Install Now → Activate
    • AliDropship dashboard will appear in the left sidebar
  • 3
    Import First Products
    • AliDropship → Products → Search Products → type GPS dog tracker
    • Browse results → click Import on desired products
    • Set title, description, images before importing
  • 4
    Configure Pricing Rules
    • AliDropship → Settings → Pricing Rules
    • Set Multiplier: 2.5 (sells at 2.5× supplier cost)
    • Set Minimum Profit: $8.00
    • Set Cents: .99 (makes prices end in .99)
  • 5
    Create Product Categories
    • WooCommerce → Products → Categories
    • Create these 15 categories: GPS Trackers, Smart Feeders, Pet Cameras, Smart Collars, Health Monitors, Pet Toys, Water Fountains, Training Devices, Grooming Tech, Pet Carriers, Safety Gear, Sleep Tech, Travel Accessories, Smart Doors, Bark Control
  • 6
    Configure Shipping Zones
    • WooCommerce → Settings → Shipping → Shipping Zones → Add Zone
    • Zone: "Canada" → Regions: Canada → add Flat Rate ($5.99) and Free Shipping ($50+ threshold)
    • Zone: "United States" → Regions: United States → add Flat Rate ($7.99) and Free Shipping ($75+)
  • 7
    Set Minimum for Free Shipping
    • WooCommerce → Settings → Shipping → Canada → Free Shipping → edit
    • Minimum order amount: 50.00

Custom Shipping Method — WooCommerce PHP Integration

To add Job Shipping or any custom carrier as a WooCommerce shipping method, create a small custom plugin with a shipping method class. This gives full control over rates, thresholds, and labels without relying on third-party shipping plugins.

File path: Create folder /wp-content/plugins/techpackcage-shipping/ and place the file below inside it as techpackcage-shipping.php
PHP — /wp-content/plugins/techpackcage-shipping/techpackcage-shipping.php
<?php
/**
 * Plugin Name: TechPackCage Custom Shipping
 * Description: Adds Job Shipping and custom rates to WooCommerce
 * Version: 1.0.0
 * Author: Francisco Holdings Inc.
 */

if ( ! defined( 'ABSPATH' ) ) exit;

function techpackcage_shipping_init() {
    if ( ! class_exists( 'WC_Shipping_Method' ) ) return;

    class TechPackCage_Shipping_Method extends WC_Shipping_Method {

        public function __construct( $instance_id = 0 ) {
            $this->id                 = 'techpackcage_shipping';
            $this->instance_id        = absint( $instance_id );
            $this->method_title       = __( 'TechPackCage Shipping', 'woocommerce' );
            $this->method_description = __( 'Custom Job Shipping rates for TechPackCage.com', 'woocommerce' );
            $this->supports           = array( 'shipping-zones', 'instance-settings' );
            $this->init();
        }

        public function init() {
            $this->init_form_fields();
            $this->init_settings();
            $this->title          = $this->get_option( 'title', 'Standard Shipping' );
            $this->base_rate      = (float) $this->get_option( 'base_rate', 5.99 );
            $this->free_threshold = (float) $this->get_option( 'free_threshold', 50.00 );

            add_action( 'woocommerce_update_options_shipping_' . $this->id,
                        array( $this, 'process_admin_options' ) );
        }

        public function init_form_fields() {
            $this->instance_form_fields = array(
                'title' => array(
                    'title'   => __( 'Method Title' ),
                    'type'    => 'text',
                    'default' => 'Standard Shipping',
                ),
                'base_rate' => array(
                    'title'       => __( 'Base Shipping Rate ($)' ),
                    'type'        => 'price',
                    'default'     => '5.99',
                    'description' => 'Flat rate charged when order is below free shipping threshold',
                ),
                'free_threshold' => array(
                    'title'       => __( 'Free Shipping Minimum ($)' ),
                    'type'        => 'price',
                    'default'     => '50.00',
                    'description' => 'Orders at or above this amount get free shipping',
                ),
            );
        }

        public function calculate_shipping( $package = array() ) {
            $cart_total = WC()->cart->get_cart_contents_total();

            if ( $cart_total >= $this->free_threshold ) {
                $rate = array(
                    'id'    => $this->get_rate_id(),
                    'label' => 'Free Shipping',
                    'cost'  => 0,
                );
            } else {
                $rate = array(
                    'id'    => $this->get_rate_id(),
                    'label' => $this->title,
                    'cost'  => $this->base_rate,
                );
            }

            $this->add_rate( $rate );
        }
    }
}
add_action( 'woocommerce_shipping_init', 'techpackcage_shipping_init' );

function add_techpackcage_shipping_method( $methods ) {
    $methods['techpackcage_shipping'] = 'TechPackCage_Shipping_Method';
    return $methods;
}
add_filter( 'woocommerce_shipping_methods', 'add_techpackcage_shipping_method' );

Activation steps:

  1. Upload folder techpackcage-shipping/ to /wp-content/plugins/
  2. WP Admin → Plugins → Activate "TechPackCage Custom Shipping"
  3. WooCommerce → Settings → Shipping → Shipping Zones → Add Zone → Add Shipping Method → select "TechPackCage Shipping"
  4. Configure base rate and free shipping threshold in zone settings
